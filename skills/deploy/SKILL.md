# Deploy Checklist Skill

## 배포 전 검증 순서

### 1. 코드 품질
- [ ] pnpm lint 통과
- [ ] pnpm build 성공 (TypeScript 컴파일 에러 없음)
- [ ] pnpm test 전체 통과

### 2. 환경 변수
- [ ] .env.example에 새 환경변수 추가 여부 확인
- [ ] 시크릿이 코드에 하드코딩되지 않았는지 확인
- [ ] 필수 환경변수 누락 시 명확한 에러 메시지 출력 확인

### 3. 데이터베이스
- [ ] 새 마이그레이션 파일이 있으면 로컬에서 테스트 완료
- [ ] 기존 데이터와 호환성 확인
- [ ] 롤백 마이그레이션 존재 여부

### 4. 외부 API
- [ ] rate limit 핸들링 적용
- [ ] 타임아웃 설정 (30초 기본)
- [ ] 서킷 브레이커 적용

### 5. Docker
- [ ] docker compose up으로 전체 서비스 기동 확인
- [ ] 헬스체크 엔드포인트 응답 확인
