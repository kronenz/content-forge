# Humanize Check Skill

## AI 스멜 단어 목록 (금지)
- "~적" 남용 (혁신적, 효율적, 전략적 등 불필요한 한자어 접미사)
- "~화" 남용 (최적화, 자동화, 체계화 등 추상적 표현)
- "다양한", "획기적인", "혁신적인"
- "~을 통해", "~에 기반하여"
- "핵심적인", "필수적인" (구체적 설명으로 대체)
- "살펴보겠습니다", "알아보겠습니다" (직접 설명으로 대체)
- 문장 시작 "또한,", "그리고,", "하지만,"

## 자연스러운 문체 규칙
- 짧은 문장 우선 (40자 이내)
- 구어체 톤 허용 (채널별 조정)
- 구체적 숫자/사례 사용
- 질문으로 시작하는 단락 허용
- 1인칭 시점 사용 (채널에 따라 "저는", "나는")

## 검증 절차
1. AI 스멜 단어 자동 검출 (정규식)
2. 문장 길이 분포 확인 (평균 30자 이내)
3. 동일 어미 반복 검출 ("~합니다" 연속 3회 이상)
4. 브랜드 바이블 톤 점수 확인
